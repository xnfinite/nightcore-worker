<h1 align='center'>Night Core Vesper  AI Assistant & Self-Documentation Layer</h1>

<p align='center'><strong>Knowledge Extraction  Reasoning Core  Autonomous Troubleshooting</strong></p>

---

##  Overview
Vesper is the self-documenting intelligence layer of Night Core. It transforms runtime data, audit logs, and module manifests into natural-language insights for developers and enterprises.

---

##  Core Components
- Knowledge Extraction Layer  parses manifests, logs, and comments.  
- Vector Embedding Layer  stores semantic data in a vector database.  
- Reasoning Core  powered by a local or external LLM.  
- Contextual Orchestrator  retrieves and assembles relevant data.  
- Interface Layer  CLI, REST API, or dashboard chat.

---

##  CLI Commands
nightcore ai explain <tenant>  
nightcore ai troubleshoot <error_id>  
nightcore ai describe <module>  
nightcore ai search <query>

---

##  Security & Audit Integration
Vesper runs under Guardian supervision and AUFS verification. Every AI output is logged and signed for traceability.

---

##  Deployment Options
- CLI mode  local assistant  
- Containerized API  
ightcore/vesper  
- Embedded Widget  dashboard chat interface
